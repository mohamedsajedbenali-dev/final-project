NekoReel is a responsive movie website showcasing films with interactive UI cards. Each movie card displays essential information like poster, title, release date, rating, description, and a button to watch the trailer. The website also includes dynamic search, genre filtering, and a dark/light theme toggle for an enhanced user experience.

---

## Project Specification

**Project Name:** NekoReel  
**Type:** Responsive Movie Website  
**Tech Stack:** HTML, CSS, JavaScript  

**Features:**
- **Movie Cards:** Display poster, title, release date, description, rating, and trailer button.  
- **Fixed Navbar:** Includes site logo, search bar, genre filter dropdown, and dark/light mode toggle.  
- **Dynamic Functionality:**  
  - Search bar filters movies by title in real-time.  
  - Genre filter displays movies of the selected category.  
  - Dark/Light theme toggle changes the website theme dynamically.  
  - Watch trailer button opens a modal with embedded YouTube trailer.  
- **Responsive Design:** Works on desktop and mobile devices using Flexbox/Grid.  
- **Footer:** Copyright information.

**Movie Data:** Stored as an array of objects in JavaScript with properties:  
`title`, `poster`, `releaseDate`, `description`, `rating`, `genre`, `trailerLink`.

**UI/UX Design:**  
- Clean, modern, and responsive design.  
- Hover effects on movie cards.  
- Smooth transitions for theme toggle and modal popups.
